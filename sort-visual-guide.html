🎨 Dizajn poboljšanja

Novi moderan dropdown dizajn:

Sort By: Price Low-High
  • Minimum širina 180px za bolje čitanje
  • Custom SVG strelica umesto native browser kontrole
  • Hover efekat sa crvenom bojom brenda (#D10024)
  • Focus state sa shadow efektom za accessibility
  • Smooth transition animacije (0.3s)
  • Zaobljeni uglovi (4px border-radius)
  • Povećan padding za mobile-friendly touch targets

💻 Tehnička implementacija

HTML struktura:

<select id="sort-select" class="input-select">
  <option value="name,asc">Name A-Z</option>
  <option value="name,desc">Name Z-A</option>
  <option value="bestRetailPrice,asc">Price Low-High</option>
  <option value="bestRetailPrice,desc">Price High-Low</option>
  <option value="id,desc">Newest</option>
  <option value="createdAt,desc">Recently Added</option>
</select>

JavaScript event handler:

$('#sort-select').on('change', function () {
  state.sort = $(this).val();  // npr. "bestRetailPrice,asc"
  state.page = 0;  // resetuj na prvu stranicu
  loadProducts();  // učitaj proizvode sa novim sortiranjem
});
